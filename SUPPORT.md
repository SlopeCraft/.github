# Support

## Choose the right channel

### Product usage and troubleshooting

For questions about using SlopeCraft or VisualCraft, configuration help, and general troubleshooting, use:

- [SlopeCraft Discussions](https://github.com/SlopeCraft/SlopeCraft/discussions)
- [SlopeCraft documentation](https://slopecraft.readthedocs.io/)

### Repository defects and feature requests

Use the issue tracker of the affected repository for:

- reproducible defects;
- repository-scoped feature requests;
- documentation errors;
- packaging or build problems.

Please search existing issues first and provide the affected version or commit, environment, reproduction steps, and relevant logs.

### Security vulnerabilities

Do not open a public issue. Follow the [security policy](https://github.com/SlopeCraft/.github/security/policy).

## Support expectations

SlopeCraft is maintained by volunteers. Support is provided on a best-effort basis, and response times are not guaranteed.

A repository-specific support document takes precedence over this default document.

# 支持说明

## 选择正确的渠道

### 产品使用与故障排查

关于 SlopeCraft 或 VisualCraft 的使用方式、配置帮助和一般故障排查，请使用：

- [SlopeCraft Discussions](https://github.com/SlopeCraft/SlopeCraft/discussions)
- [SlopeCraft 文档](https://slopecraft.readthedocs.io/)

### 仓库缺陷与功能请求

以下内容请提交到受影响仓库的 Issue：

- 可重现的缺陷；
- 限定于当前仓库的功能请求；
- 文档错误；
- 打包或构建问题。

请先搜索已有 Issue，并提供受影响版本或提交、运行环境、重现步骤和相关日志。

### 安全漏洞

不要创建公开 Issue，请遵循[安全策略](https://github.com/SlopeCraft/.github/security/policy)。

## 支持预期

SlopeCraft 由志愿者维护，仅提供尽力支持，不保证响应时间。

如果仓库内提供了自己的支持说明，应以仓库内文件为准。
